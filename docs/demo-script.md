# 1 分钟演示话术

大家好，这是我为塔外智能设计的官网重构作品：TowerOS 企业智能体应用平台官网。

我没有把它做成普通公司介绍页，也没有做成过度炫技的 AI 页面，而是采用更接近大公司 B2B 官网的浅色科技风，突出企业级、可信和可落地。

首页表达核心定位：让智能体进入真实业务场景，成为可执行、可复核、可沉淀的数字工作单元。产品页展示岗位任务智能体、企业知识智能体和业务流程智能体。场景页展示工业安全、运营复盘、文档生成和岗位任务矩阵。

最重要的是方案顾问页。访客可以输入“我们公司想做工业安全巡检”，系统会推荐工业安全智能体，并输出业务痛点、核心能力、落地路径和预期价值。

后端使用 Python FastAPI，实现了对话接口、知识库检索和线索保存。第一版采用规则流保证演示稳定，后续可以接入大模型 API 和企业知识库，变成真实可部署的智能官网入口。
