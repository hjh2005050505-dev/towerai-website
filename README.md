# 塔外智能 TowerOS 官网与 AI Agent 入口

这是一个面向 HR/老板展示的产品型官网提案作品：不是普通公司介绍页，而是“塔外智能官网 + AI Agent 智能入口 + 企业智能体产品演示平台”。

## 核心亮点

- 深色科技感品牌首页
- TowerOS 智能中枢动态视觉
- 产品体系：岗位任务智能体、企业知识智能体、业务流程智能体
- 场景案例：工业安全、运营复盘、文档生成、岗位任务矩阵
- AI Agent：方案推荐、方案摘要、预约线索收集
- Python FastAPI 后端 + SQLite 线索保存

## 启动方式

1. 进入项目目录：work/towerai_website
2. 安装依赖：pip install -r requirements.txt
3. 启动服务：uvicorn app.main:app --reload
4. 打开：http://127.0.0.1:8000
